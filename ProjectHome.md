# DBCtoSQL v0.4 #

the DBC to SQL Converter

## README ##
https://dbctosql.googlecode.com/hg/README

### INFO ###
Program can convert DBC files to SQL files.

So folks could use the SQL language's benefits.

The program written for people who want do some (re)searcing in DBC files like me ;)

**DBC supports:**
  * Spell.dbc - v3.1.3(9947)
  * Achievement.dbc - v3.1.3(9947)
  * Achievement\_Category.dbc - v3.1.3(9947)
  * Achievement\_Criteria.dbc - v3.1.3(9947)
